HQ Radar for GTA: San Andreas (author: yojo2) v1.02
I spent long time making this mod, so please don't steal my work. (or else you'll be rickrolled)

Installation:
- using IMG Tool: open gta3.img and replace every txd named radarXX.txd to those shipped with the mod. You can backup original files in case you don't like my mod.
OR:
- using San Andreas Mod Installer: double click _hqr-sa1.sami file and install it. Then do the same with _hqr-sa2.sami file. SAMI will install the mod for you and will backup original files too.

Map was made in Iconscape.
